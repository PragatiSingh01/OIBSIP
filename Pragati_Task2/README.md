# BMI Calculator

## Description
A simple BMI Calculator built using Python and Tkinter. It allows users to enter their weight and height, calculates BMI, and displays the health category.

## Features
- User-friendly GUI
- BMI Calculation
- BMI Classification
- Input Validation
- Error Handling

## Technologies Used
- Python
- Tkinter

## BMI Formula
BMI = Weight (kg) / Height² (m²)

## How to Run
1. Install Python.
2. Download the project.
3. Run:
   python bmi_calculator.py
