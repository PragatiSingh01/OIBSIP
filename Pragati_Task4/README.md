# Prags_Task4 - Basic Weather App

## Project Description
This project is a simple Weather Application developed using Python. It fetches real-time weather information through API integration.

## Features
✔ Temperature display  
✔ Humidity display  
✔ Weather condition display  
✔ Wind speed display  
✔ Error handling  
✔ User input validation  

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## Installation

Install dependencies:

pip install -r requirements.txt

## Run Project

python Prags_Task4.py


## API Setup

1. Visit:
https://openweathermap.org/api

2. Create free account

3. Copy API key

4. Replace:

API_KEY = "YOUR_API_KEY"

with your actual API key.
