# Voice Assistant - Task 1

## Description
A simple voice assistant built using Python that can:
- Respond to greetings
- Tell current time
- Tell current date
- Open Google
- Exit on voice command

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- PyAudio

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python voice_assistant.py
```

## Commands
- Hello
- Time
- Date
- Google
- Bye / Stop / Exit
